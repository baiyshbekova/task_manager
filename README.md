# Task Manager (To-Do List)

## Консольное приложение на C# для управления списком задач

## Функциональность
- Добавление задачи
- Просмотр списка задач
- Отметка задачи как выполненной
- Удаление задачи
- Сохранение задач в файл json

## Структура проекта
- Program.cs — меню приложения
- TaskItem.cs — описание одной задачи, ее номер, содержание и статус выполнения (Id, Title, IsCompleted)
- TaskManager.cs — логика управления задачами и сохранения данных (добавление, удаление, выполнение задачи, просмотр)
